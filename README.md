# Customer Purchase Prediction and Segmentation Using Machine Learning

## Project Overview

This project analyzes customer purchasing behavior using Machine Learning techniques. The objective is to predict whether a customer is a high spender or low spender and to segment customers into different groups based on their purchasing patterns.

## Features

* Data Preprocessing
* Feature Scaling
* PCA Visualization
* Classification Algorithms

  * Logistic Regression
  * KNN
  * Naive Bayes
  * Decision Tree
  * Random Forest
  * SVM
  * Gradient Boosting
  * XGBoost
* KMeans Clustering
* Hierarchical Clustering
* Confusion Matrix
* Feature Importance Analysis
* ROC Curve and AUC Score
* Correlation Heatmap
* Customer Prediction System

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* XGBoost

## Dataset

Marketing Campaign Dataset

## Project Outcome

The project helps identify customer spending patterns, perform customer segmentation, and predict high-value customers for better business decision-making.
